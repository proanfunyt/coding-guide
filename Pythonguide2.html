Guide: How to Code in Python

Python Functions

Functions help you organize Python code into reusable pieces. Instead of writing the same instructions more than once, you define a function once and call it whenever you need it.

def greet(name):
    message = "Hello, " + name + "!"
    print(message)

# Run the function with a specific name
greet("Antonio")

How it works:

  • def starts the function definition.
  • greet is the function name.
  • (name) is the parameter: a placeholder for the input you give the function.
  • Indented lines inside the function are the actions Python performs when the function runs.

Calling greet("Antonio") sends the value "Antonio" into the function, so the code inside can use it.

Working with Lists

Lists store multiple values in one variable. Each item in a list keeps its order, and you can use an index to access them.

fruits = ["apple", "banana", "cherry"]
print(fruits[0])  # apple

# Add a new item to the list
fruits.append("orange")

# Loop over each item in the list
for fruit in fruits:
    print(fruit)

How it works:

  • ["apple", "banana", "cherry"] creates a list with three items.
  • fruits[0] gets the first item, because Python starts counting at zero.
  • append() adds a new value to the end of the list.
  • for fruit in fruits: repeats the same action for every item in the list.

Dictionaries

Dictionaries let you store information using named keys instead of numeric positions. This makes data easier to read and use.

person = {
    "name": "Antonio",
    "age": 20,
    "language": "Python"
}
print(person["name"])

# Change a value
person["age"] = 21
print(person["age"])

# Loop through keys and values
for key, value in person.items():
    print(key + ":", value)

How it works:

  • {"name": "Antonio", "age": 20} creates a dictionary.
  • Each key (like "name") points to a value (like "Antonio").
  • person["name"] looks up the value for the key "name".
  • person.items() returns all key/value pairs so you can loop over them.

Useful Python Built-Ins

Built-in functions are ready to use without importing anything. They help you do common tasks quickly.

  • len(value): Returns how many items are in a string, list, tuple, or dictionary.
  • type(value): Tells you the data type of a value.
  • input(prompt): Shows a message, waits for keyboard input, and returns the typed text.
  • str(), int(), float(): Convert values to text, whole numbers, or decimal numbers.
name = input("What is your name? ")
print("Nice to meet you, " + name)
print("Your name has", len(name), "letters.")

age_text = input("How old are you? ")
age = int(age_text)
print("Next year you will be", age + 1)

How it works:

  • input() pauses the program and waits for the user to type something.
  • len(name) counts characters in the typed name.
  • int(age_text) converts the text input into a number so you can add to it.

Python Modules

Modules are collections of code that Python or other people already wrote. By importing a module, you can use its tools instead of writing them yourself.

import math
print(math.sqrt(25))  # 5.0
print(math.pi)        # 3.141592653589793

import random
colors = ["red", "blue", "green"]
print(random.choice(colors))

How it works:

  • import math makes the math module available.
  • math.sqrt(25) calls the sqrt function from that module.
  • random.choice(colors) picks a random item from the list.
  • Modules group useful tools so your code stays smaller and easier to read.

Why This Matters

Understanding functions, lists, dictionaries, built-ins, and modules is the foundation of writing useful Python programs. These tools let you:

  • Break code into reusable pieces with functions so it is easier to understand and maintain.
  • Store and process groups of values with lists and dictionaries to handle real-world data.
  • Interact with users and convert data types using built-in functions.
  • Reuse powerful existing code by importing modules instead of building everything from scratch.

Once you master these concepts, you can build small programs, solve problems faster, and move on to more advanced Python topics with confidence.

Next Steps

Try practicing with a few simple projects to make these ideas stick. For example:

  • Write a program that asks for a name and age, then greets the user and calculates their age next year.
  • Create a list of your favorite foods and print each item with a sentence.
  • Build a dictionary to store a friend's contact details and print the values in a friendly format.

After that, move on to learning functions with return values, file input/output, and error handling.
