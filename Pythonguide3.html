Guide: How to Code in Python

Functions with Return Values

Functions can send a value back to the part of your program that called them. This is useful when you want the result of a calculation or a transformed value.

def add(a, b):
    return a + b

result = add(5, 3)
print(result)  # 8

How it works:

  • return sends a value back to the caller.
  • The function stops running after it reaches return.
  • result receives the value produced by add(5, 3).

File Input / Output

File I/O lets Python read and save information from files on your computer. This is how you keep data after the program stops.

with open("notes.txt", "w") as file:
    file.write("Hello from Python!\n")

with open("notes.txt", "r") as file:
    content = file.read()
    print(content)

How it works:

  • open("notes.txt", "w") opens a file for writing, creating or replacing it.
  • with ensures the file is closed automatically when you are done.
  • file.write() sends text into the file.
  • open("notes.txt", "r") opens the same file for reading.
  • file.read() gets the file contents as one string.

Error Handling

Error handling helps your program stay running even when something goes wrong, like bad input or a missing file.

try:
    age_text = input("Enter your age: ")
    age = int(age_text)
    print("Next year you will be", age + 1)
except ValueError:
    print("Please type a valid number.")
finally:
    print("Program finished.")

How it works:

  • try runs code that might cause an error.
  • except runs only if that error happens.
  • ValueError is a common error when a conversion like int() fails.
  • finally always runs, even if there was an error.

Why These Topics Matter

Return values, file I/O, and error handling are important because they let your programs do more useful work:

  • Functions with returns allow your code to calculate results and use them later.
  • File I/O saves and loads data so your program can remember things between runs.
  • Error handling makes your program safer and easier to use by preventing crashes.
