Guide: How to Code in Python

Introduction to Python

Welcome to the Python guide! Python is a popular programming language known for its simplicity and readable style. It is used for web development, data analysis, automation, machine learning, and much more. Think of Python as a powerful tool that lets you tell a computer what to do using English-like commands.

Python is a true programming language because it can perform calculations, make decisions, and repeat actions automatically. It uses code instead of markup tags, making it a great choice for beginners and experienced developers alike.

Basic Python Concepts

Python code is executed from top to bottom. A Python program is made up of statements, expressions, and values. Here are some of the most important building blocks:

  • Variables: Containers for storing values, such as text or numbers. Example: name = "Proanfun".
  • Data types: Common types include str for text, int for whole numbers, float for decimals, and bool for True/False values.
  • Functions: Named blocks of code that perform tasks. Example: print() shows text on the screen.
  • Comments: Notes for humans that Python ignores. Use # to start a comment.

Python is indentation-sensitive:

Use consistent spacing or tabs to indent code blocks. Indentation tells Python what belongs together, such as the lines inside an if statement or a loop.

Writing Your First Python Program

Here is a simple Python script that prints a greeting and performs a small calculation:

name = "Proanfun"
print("Hello, " + name + "!")

age = 20
next_year = age + 1
print("Next year you will be", next_year)

What is happening?

  1. name = "Proanfun": Stores the text "Proanfun" in a variable named name.
  2. print("Hello, " + name + "!"): Joins the greeting text and the name, then displays it in the console.
  3. age = 20: Stores the number 20 in a variable named age.
  4. next_year = age + 1: Adds 1 to the age and stores the result in next_year.
  5. print("Next year you will be", next_year): Shows the result in the console.

Python Control Flow

Python can make decisions using if statements and repeat actions using loops.

x = 5
if x > 0:
    print("x is positive")
else:
    print("x is zero or negative")

for i in range(3):
    print("Number", i)

How it works

  • if x > 0: checks whether the value of x is greater than zero.
  • The indented lines below if or else only run when that branch is true.
  • for i in range(3): repeats the loop three times, using the values 0, 1, and 2.
  • Loops are useful for repeating actions without writing the same code again.

Common Python Elements

These are the most useful parts of Python to know first:

  1. Strings: Text inside quotes, such as "Python". Strings can be joined or measured with len().
  2. Numbers: Whole numbers are int, decimals are float, and they can be used in math operations.
  3. Lists: Ordered collections of values. Example: [1, 2, 3]. Use indexes like my_list[0] to get the first item.
  4. Dictionaries: Collections of named values using keys. Example: {"name": "Proanfun", "age": 20}. Use person["name"] to look up a value.
