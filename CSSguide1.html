Guide: How to Code in CSS

What is CSS?

CSS stands for Cascading Style Sheets. It controls how HTML elements look on the page by defining colors, fonts, spacing, layout, and more. While HTML creates the structure, CSS makes it beautiful and readable.

CSS is applied to HTML using selectors that choose which elements to style, then properties that define the visual rules.

CSS Syntax

Every CSS rule has a selector, a property, and a value. Together they tell the browser what to style and how.

h1 {
    color: #38bdf8;
    font-size: 2rem;
    margin-top: 0;
}

How it works:

  • h1 is the selector that targets heading elements.
  • color is the property.
  • #38bdf8 is the value that sets the heading color.

Common Selectors

Selectors tell CSS which elements to style. Here are the most useful ones:

  • element selects all tags of that type, like p or div.
  • .class selects elements with a specific class attribute.
  • #id selects one unique element with an id.
  • element.class combines tag and class for more precise selection.
.button {
    background-color: #38bdf8;
}

#main-title {
    font-size: 3rem;
}

p.lead {
    color: #94a3b8;
}

The Box Model

Every HTML element is a rectangular box. The CSS box model controls the element's size and spacing using four layers.

box-sizing: border-box;
padding: 20px;
border: 2px solid #334155;
margin: 20px;

The four layers are:

  • Content: The actual text or image inside the element.
  • Padding: Space between the content and the border.
  • Border: The line around the element.
  • Margin: Space outside the border that separates elements.

Typography, Colors, and Backgrounds

CSS makes text readable and gives your page personality. Use fonts, colors, and backgrounds to create a polished design.

body {
    font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
    background-color: #0f172a;
    color: #f8fafc;
}

h2 {
    color: #38bdf8;
}

section {
    background: #1e293b;
    border-radius: 8px;
}

Use CSS to control:

  • font-family for text style.
  • color for text color.
  • background-color for element backgrounds.
  • border-radius to round corners.

Layout Basics

Layout is how elements are arranged on the page. These CSS properties are essential for modern layouts:

.container {
    display: flex;
    gap: 20px;
}

.box {
    flex: 1;
    padding: 20px;
}

.header {
    display: grid;
    place-items: center;
}

Key layout tools:

  • display: Changes how an element behaves, such as block, inline, flex, or grid.
  • flex: Creates flexible row or column layouts.
  • grid: Builds precise two-dimensional layouts.
  • gap: Adds space between flex or grid items.

Why CSS Matters

CSS turns plain HTML into a visually appealing experience. It helps your website:

  • look professional and polished,
  • create readable layouts for users,
  • stay consistent across pages,
  • adapt to different screen sizes with responsive design.

Next Steps

Practice by styling a simple webpage. Try these challenges:

  • Create a card with a colored background, padding, and rounded corners.
  • Use a class selector to style several buttons the same way.
  • Build a simple flex layout with two or three boxes side by side.

After that, explore responsive design with media queries and CSS transitions for smooth animations.
