Guide: How to Code in JavaScript

JavaScript in the Browser

JavaScript is commonly used inside web pages to change content, respond to user input, and update the page without reloading. The browser provides tools like document and window to let your code interact with the page.

console.log("JavaScript is running!");
console.log(document.title);

How it works:

  • console.log() prints messages to the browser console.
  • document.title reads the current page title.
  • The browser executes JavaScript while the page is open.

Events and Interaction

Events occur when a user clicks a button, types text, or moves the mouse. JavaScript responds to these events with event listeners.

const button = document.querySelector("button");
button.addEventListener("click", function() {
    console.log("Button clicked!");
});

How it works:

  • querySelector() finds an element on the page.
  • addEventListener() listens for an action like a click.
  • The function inside runs when the event happens.

DOM Manipulation Basics

The DOM is the browser's representation of the page. JavaScript can read and change DOM elements to update text, styles, and structure.

const title = document.querySelector("h1");
title.textContent = "Hello from JavaScript!";

document.body.style.backgroundColor = "#0f172a";

How it works:

  • textContent updates visible text on the page.
  • style.backgroundColor changes an element's style.
  • JavaScript can modify the page after it has loaded.

Array Methods

JavaScript arrays have built-in methods for common tasks such as looping, filtering, and transforming values.

const numbers = [1, 2, 3, 4];
const squared = numbers.map(function(n) {
    return n * n;
});
const evens = numbers.filter(function(n) {
    return n % 2 === 0;
});
console.log(squared);
console.log(evens);

How it works:

  • map() creates a new array by changing each item.
  • filter() creates a new array with only the items that match a condition.
  • These methods keep code shorter and easier to read.

Debugging with the Console

The browser console helps you see errors and inspect values while your code runs. It is one of the best tools for learning and troubleshooting JavaScript.

const greeting = "Hello";
console.log(greeting);
console.error("This is an error message.");
console.table([{ name: "JS", level: "easy" }]);

How it works:

  • console.log() shows normal messages.
  • console.error() shows errors and makes them easier to spot.
  • console.table() displays arrays or objects in a table layout.
