Guide: How to Code in HTML

Introduction to HTML

Welcome to my guide on coding with HTML! HTML stands for HyperText Markup Language. It is the absolute backbone of the internet. Every single website you visit—from YouTube to Google—uses HTML to structure its content. Think of HTML as the skeleton of a building; it provides the structure, while CSS provides the paint and decorations.

HTML isn't a programming language like Python or C# because it doesn't handle logic or calculations. Instead, it is a markup language, meaning it uses special tags to flag text so a web browser knows whether to display it as a heading, a paragraph, a button, or an image.

Understanding HTML Tags

HTML relies completely on components called Elements, which are wrapped inside Tags. Tags use angle brackets (< and >) to tell the browser what to do.

Almost all tags come in pairs: an opening tag and a closing tag. The closing tag looks exactly like the opening tag but features a forward slash (/) right before the tag name.

Anatomy of a Tag:

<p> This is how you open a paragraph. </p> This is how you close it.

If you forget to close a tag, the browser will get confused and might accidentally apply that styling to the entire rest of your webpage!

The Essential Page Structure

Every single standard HTML document requires a specific structure to load correctly in web browsers. Here is the absolute template breakdown that every developer uses:

<!DOCTYPE html>
<html>
<head>
    <title>Page Title Displayed on Tab</title>
</head>
<body>
    <h1>This is a Main Heading</h1>
    <p>This is where the text goes.</p>
</body>
</html>

What do these mean?

  • <!DOCTYPE html>: Tells the browser that this file is written in modern HTML5.
  • <html>: The root element that wraps absolutely everything on the page.
  • <head>: Contains background info that users don't see, like the page title or fonts.
  • <body>: Contains all visible text, pictures, buttons, and layouts that users actually interact with.

The Most Important HTML Elements

Once your body structural tag is set up, you can populate it using a variety of basic elements:

  1. Headings: Ranging from <h1> (largest, most important) down to <h6> (smallest sub-heading).
  2. Paragraphs: Handled with the <p> tag for regular readable body text block.
  3. Links: Programmed via the anchor <a> tag. It requires an attribute called href to tell it where to point, like this:
    <a href="https://google.com">Visit Google</a>
  4. Lists: Use <ol> for numbered (ordered) lists, <ul> for bulleted (unordered) lists, and place <li> tags inside them for each item.
