Hidden Guide

This page is only visible when opened directly by file path.

What this page contains

This hidden page follows the same dark theme and layout used by your HTML, Python, and CSS guides. It is intentionally not linked from the rest of the site, so it is only accessible if someone opens HiddenGuide.html directly.

The page uses the same structure: a header, content sections, code examples, and explanation notes. It also shows you exactly how the theme is built so you can write your own page.

Page structure explained

Every page has the same basic HTML structure. Use this as your starting template:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Your Page Title</title>
    <style>...styles...</style>
</head>
<body>
    <div class="container">
        ...page content...
    </div>
</body>
</html>

Key HTML pieces:

  • <!DOCTYPE html>: Declares HTML5 so browsers render the page correctly.
  • <html lang="en">: Starts the HTML document and sets the language to English.
  • <head>: Holds metadata such as the page title and internal CSS.
  • <style>: Contains all of the page's CSS styling.
  • <body>: Contains everything the visitor sees.

Theme and styles

The theme is built with CSS variables and consistent card styling. Update the variables to change the entire look of your page without rewriting every rule.

:root {
    --bg-color: #0f172a;
    --card-bg: #1e293b;
    --text-color: #f8fafc;
    --text-muted: #94a3b8;
    --accent-color: #38bdf8;
    --accent-hover: #7dd3fc;
    --border-radius: 8px;
}

These variables are used in the main style rules for the body, cards, headings, buttons, and other elements. For a new page, keep this section and only change the values you want.

Tip: To use the same theme on a new page, copy the entire :root block and the CSS rules for body, .container, section, h1, .subtitle, and .btn.

Visible page sections

Create content using multiple <section> blocks. Each section should have a title and a few paragraphs to explain the topic.

<section>
    <h2>Section Title</h2>
    <p>This is the explanation for the section.</p>
</section>

If you want to show code examples, wrap the sample in <pre><code> so the browser preserves spacing.

How code blocks work

The <pre> tag preserves the formatting of the code. The nested <code> tag gives the text the correct font and color. This is the best way to display code samples cleanly.

<pre><code>
# example code
print("Hello")
</code></pre>

You can use the same CSS styling on new pages by copying the rules for pre and code from this page.

Build your own page step-by-step

Follow these steps to create a new page with the same dark theme:

  1. Copy the whole page from this file, including the <style> block.
  2. Change the title inside <title> and the top heading inside <h1>.
  3. Add new <section> boxes with your own headings and text.
  4. If you want the page hidden, do not add links to it from other pages.
  5. Save it with a new name like MyNewGuide.html.

Important: No links from other pages means the page stays hidden.
