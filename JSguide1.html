Guide: How to Code in JavaScript

Introduction to JavaScript

JavaScript is the programming language that makes web pages interactive. While HTML gives structure and CSS adds style, JavaScript allows a page to respond to clicks, update content, and make things happen automatically.

JavaScript runs in the browser and is used for games, forms, animations, and web applications. It is a powerful language that works well for beginners because it can show results immediately in the browser.

Variables and Data Types

Variables store values that your program can use later. JavaScript supports different data types like text, numbers, and true/false values.

let name = "Proanfun";
const age = 20;
let isStudent = true;
let total = 5 + 3;

How it works:

  • let creates a value that can change later.
  • const creates a value that cannot be reassigned.
  • "Proanfun" is a string, 20 is a number, and true is a boolean.
  • JavaScript can do math, store text, and remember values in variables.

Functions and Expressions

Functions are reusable blocks of code. You define a function once and call it whenever you need the same work done again.

function greet(name) {
    console.log("Hello, " + name + "!");
}

greet("Proanfun");

How it works:

  • function greet(name) defines a function named greet with a parameter.
  • console.log() shows text in the browser console.
  • greet("Proanfun") runs the function and passes in the value "Proanfun".

Control Flow

Control flow decides which code runs when. JavaScript uses if statements and loops to make choices and repeat actions.

let score = 8;
if (score >= 5) {
    console.log("You passed!");
} else {
    console.log("Try again.");
}

for (let i = 0; i < 3; i++) {
    console.log("Step", i + 1);
}

How it works:

  • if (score >= 5) checks whether the score is high enough.
  • Code inside the curly braces runs only when the condition is true.
  • for loops repeat code multiple times using a counter.

Arrays and Objects

Arrays store ordered lists of values, and objects store named values. They're both essential for organizing data in JavaScript.

let colors = ["red", "green", "blue"];
console.log(colors[0]);

let user = {
    name: "Proanfun",
    age: 20,
    active: true
};
console.log(user.name);

How it works:

  • ["red", "green", "blue"] creates an array with three items.
  • colors[0] retrieves the first item.
  • An object uses keys like name to store values.
  • user.name looks up the value for that key.
