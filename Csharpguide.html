C# Coding Guide

Variables and Data Types

Variables are containers used to store data values. C# is a strongly-typed language, meaning you must declare the data type explicitly.

int userAge = 21;
string userName = "Alice";
double totalScore = 95.5;
bool isCoding = true;

Key highlights:

  • int stores whole numbers (integers).
  • string stores text values inside double quotes.
  • double holds decimal precision values.
  • bool stores conditional flags: true or false.

Conditional Logic

Use decision-making statements like if, else if, and else to execute different paths of code.

int score = 85;

if (score >= 90) {
    Console.WriteLine("Grade: A");
} else if (score >= 80) {
    Console.WriteLine("Grade: B");
} else {
    Console.WriteLine("Grade: C");
}

Key highlights:

  • The code inside braces runs only if the condition evaluates to true.
  • Console.WriteLine() outputs strings directly onto the screen terminal.

Loops (For Loops)

Loops allow you to execute a specific block of code repeatedly over a set range or collection.

for (int i = 0; i < 5; i++) {
    Console.WriteLine("Iteration number: " + i);
}

Key highlights:

  • int i = 0 sets up the counter initialization.
  • i < 5 defines the boundary conditional statement loop criteria.
  • i++ increments your loop counter by exactly one on every lap.

Methods (Functions)

Methods isolate reusable code pieces. They can receive arguments and return calculated results.

public int AddNumbers(int num1, int num2) {
    return num1 + num2;
}

// How to call it:
int result = AddNumbers(5, 10);

Key highlights:

  • public allows outside scopes to discover the function structure safely.
  • The first type declaration (int) dictates the return outcome type.
  • return stops calculations and transmits value directly out.
