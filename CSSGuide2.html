Advanced CSS Topics

Responsive Design

Responsive design helps your website look great on phones, tablets, and desktops. Use media queries to change styles based on screen width.

@media (max-width: 768px) {
    .container {
        padding: 15px;
    }

    .nav-buttons {
        flex-direction: column;
    }
}

How it works:

  • @media checks the screen size or device features.
  • Styles inside the block only apply when the condition is true.
  • This makes layouts flexible and easier to use on small phones.

CSS Transitions

Transitions animate property changes smoothly when a user interacts with an element.

.btn {
    background-color: #38bdf8;
    transition: background-color 0.3s ease, transform 0.2s ease;
}

.btn:hover {
    background-color: #7dd3fc;
    transform: translateY(-2px);
}

How it works:

  • transition defines which properties animate and how long they take.
  • Property changes happen smoothly instead of instantly.
  • This improves the user experience by making interactions feel polished.

CSS Animations

Animations let you move and change elements over time. They are defined using @keyframes.

@keyframes float {
    0% {
        transform: translateY(0);
    }
    50% {
        transform: translateY(-10px);
    }
    100% {
        transform: translateY(0);
    }
}

.card {
    animation: float 4s ease-in-out infinite;
}

How it works:

  • @keyframes defines the animation steps.
  • animation applies it to an element.
  • infinite makes it repeat forever.

Transformations

Transforms rotate, scale, or move elements without changing the document layout.

.card:hover {
    transform: scale(1.02) rotate(-1deg);
}

.icon {
    transform: translateX(10px);
}

Common transform functions:

  • translate() moves an element horizontally or vertically.
  • scale() changes the size.
  • rotate() turns the element around a point.

Pseudo-Classes and Pseudo-Elements

Pseudo-classes and pseudo-elements target special states or parts of elements without adding extra HTML.

a:hover {
    color: #7dd3fc;
}

button:focus {
    outline: 2px solid #38bdf8;
}

button::after {
    content: " →";
}

How it works:

  • :hover applies styles when the mouse is over an element.
  • :focus styles an element when it is selected by keyboard or click.
  • ::after inserts content after an element.

CSS Variables

CSS variables store reusable values for colors, spacing, and more.

:root {
    --main-bg: #0f172a;
    --main-color: #f8fafc;
    --accent: #38bdf8;
}

body {
    background-color: var(--main-bg);
    color: var(--main-color);
}

.button {
    background-color: var(--accent);
}

How it works:

  • --name defines a variable.
  • var(--name) uses it.
  • Variables make themes and color changes easier across the whole page.

Why Advanced CSS Matters

Advanced CSS helps your website feel modern and polished. It improves usability, makes layouts flexible, and adds motion that guides the user.

  • Responsive design ensures content works on every screen.
  • Transitions and animations make interactions feel smooth.
  • Transforms add subtle movement and emphasis.
  • Pseudo-classes keep code clean while controlling states.
