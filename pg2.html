Guide: Advanced HTML Concepts

1. Mastering Image Attributes

To display an image in HTML, you use the <img> tag. Unlike paragraphs or headings, the image tag is a self-closing tag, meaning it doesn't need a closing </img> component.

Instead, it uses key identifiers called attributes to function properly:

<img src="pc.jfif" alt="Gaming Setup Setup" width="300" height="200">

Attribute Breakdown:

  • src (Source): The path or URL telling the browser exactly where the image file lives.
  • alt (Alternative Text): Text that displays if your image breaks, and reads aloud to visually impaired users. It is essential for professional design!
  • width & height: Sets pixel boundaries so your webpage layouts don't shift awkwardly while loading.

2. Structuring Data with Tables

When you want to showcase structured columns and rows—like a computer build parts list or school timetable—HTML offers the <table> element system.

<table>
    <tr>
        <th>Subject</th>
        <th>Difficulty</th>
    </tr>
    <tr>
        <td>ICT</td>
        <td>Fun / Medium</td>
    </tr>
</table>

How Table Elements Connect:

  • <tr> (Table Row): Wraps every individual line container across the grid.
  • <th> (Table Header): Defines bold, descriptive header titles at the very top of each column.
  • <td> (Table Data): Handles standard, standard row grid cells containing text.

3. Creating Interactive Forms

To accept input data from users—like building a login form, search engine bar, or comment section—you wrap structural items in a <form> tag using interactive elements:

<form>
    <label>Enter Your Username:</label>
    <input type="text" placeholder="e.g., CodeMaster123">
    <button type="submit">Send Data</button>
</form>
